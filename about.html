TeaRounder is designed to be the fastest and most efficient way to note down a drinks round list, that you can use anywhere.

You don't need any signal or WiFi to use it, it's built for phones and can also be used on tablets. It's simple to use, and you don't need to log in to anything. Android is the first platform that TeaRounder will be released for, iOS coming soon (more info in the coming soon section). The recommended way to get going with TeaRounder is to play around with it.The help and tricks section is handy if you want to find quicker ways of doing things

Whether you're being the perfect host, or you're out somewhere that serves drinks at the bar; you don't want to have to come back to the group with questions.

Instead you can use TeaRounder, and with a few taps every detail you're likely to need will be listed neatly in the app.

There is more planned for the app. New features will be added every few months, and a list of planned features can be found in the coming soon section. You can just get on with using the ap

So far:

This version is the first release, with the fundamental part of the vision for the app: a simple and sleek way to create a drinks list, with lots of popular drinks available to choose from, plus the ability to add other drinks to the list.

Future updates:

Features planned for the next few releases: ability to set your own drinks selection, customizable colour scheme and fonts, dedicated option to note down who each drink is for (before this is added the best way to record this is in the additional notes for each drink line), ability to save and load drinks lists - useful when everyone wants their usual or the same again.
